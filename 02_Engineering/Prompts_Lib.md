# Prompts Library

## Development Prompts

### Code Generation
```
Створи [компонент/функцію] для [опис функціональності] використовуючи [технологія]
```

### Code Review
```
Переглянь цей код на предмет:
- Безпеки
- Продуктивності
- Читабельності
- Best practices
```

### Debugging
```
Допоможи знайти і виправити помилку в цьому коді: [код]
Симптоми: [опис проблеми]
```

## Design Prompts

### UI/UX
```
Створи дизайн [компонент] який:
- Відповідає нашому Visual Style
- Зручний для користувача
- Адаптивний
```

## Documentation Prompts

### Technical Docs
```
Створи документацію для [функція/API] включаючи:
- Опис
- Параметри
- Приклади використання
- Можливі помилки
```
